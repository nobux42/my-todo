# Project: プロジェクト YY（ダミー）

## Goal
- 架空プロジェクトとして、要件定義〜実行〜ふりかえりまでの運用を「テンプレ化」し、次のプロジェクトに再利用できる形に整える

## Next Actions
- [ ] プロジェクトの背景と狙いを3行で言語化（Why/What/Who）
- [ ] ステークホルダーと期待値を整理（誰が何を期待しているか）
- [ ] 主要アウトプットを定義（成果物、フォーマット、期限）
- [ ] マイルストーンを4つに分ける（調査→設計→実装→検証）
- [ ] 週次の運用ルールを決める（レビュー、意思決定、共有の場）
- [ ] 「やらないこと」を明記してスコープを固定する
- [ ] リスク登録簿を作る（発生確率×影響、対策、オーナー）
- [ ] 初回の小さな検証（30〜60分）を実施して前提を検証する
- [ ] ふりかえりテンプレを用意（Keep/Problem/Try + 次の一手）

## Notes
- 目標は“完璧な計画”ではなく、“次も回せる型”を残すこと
